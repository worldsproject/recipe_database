How to Use

There are 3 differnt types of requests you can make to get recipies back.

Search by id

https://db.yourdailymenu.org/api/v1/recipes/id?key=yourkey

Any other recipe result also returns a recipe id, which can be used here to retreive an individual recipe.

Search by name

https://db.yourdailymenu.org/api/v1/recipes

POST Data

  • name : what you are looking for
  • key : your api key

Search by ingredient

https://db.yourdailymenu.org/api/v1/ingredients

POST Data

  • with : Ingredients your recipe must have.
  • without : Ingredients your recipe must not have.
  • key : your api key
 Each with and without item should be submitted as seperate variables. Submit a POST request like so:
https://db.yourdailymenu.org/api/v1/ingredients
POST Data
key:yourkey
with:onion
with:garlic
without:tomato
